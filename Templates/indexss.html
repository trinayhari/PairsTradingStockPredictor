About this project

Our project utilizes the pairs trading strategy to analyze stock trends across different sectors. Using financial data from companies in Technology, Materials, Financials, Energy, and Healthcare, we calculate correlation coefficients to identify highly correlated stock pairs. The user-friendly web application, built with Flask, allows investors to select a sector and discover potentially profitable trading opportunities based on data-driven analysis. With our tool, traders can make informed investment decisions and optimize their strategies effectively.

Analyze stocks within different sectors

This application is a Python web application that helps users identify highly correlated stock pairs within different sectors. It utilizes the yfinance library to fetch historical stock data for companies listed in various sectors like 'Tech,' 'Materials,' 'Financials,' 'Energy,' and 'Healthcare.' The Flask framework is used to create a simple web interface where users can select a sector from a dropdown menu. Upon submission, the application calculates the correlation between each pair of stocks within the chosen sector and displays the pair with the highest correlation. The tool is valuable for investors interested in pairs trading strategies, as it facilitates the identification of potential stock pairs that exhibit strong price relationships.
